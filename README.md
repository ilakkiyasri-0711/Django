Task Management Project
Overview

This is a web-based Task Management application built using Python and Django.
The project helps users create, update, and manage daily tasks efficiently.

Features

Create new tasks

Edit existing tasks

Delete tasks

Mark tasks as completed

Simple and user-friendly interface

SQLite database integration

Technologies Used

Python

Django

SQLite3

HTML / CSS (Frontend)

Project Structure

manage.py – Django command-line utility

db.sqlite3 – Default SQLite database

settings.py – Project configuration

urls.py – URL routing

models.py – Database models

views.py – Application logic

Installation

Clone the repository

Create a virtual environment

Install dependencies

pip install django


Run migrations

python manage.py migrate


Start the server

python manage.py runserver

Usage

Open your browser and go to:

http://127.0.0.1:8000/


You can now add and manage tasks.
